Etiquetas HTML

Este ejemplo muestra cómo combinar algunas de las etiquetas mas basicas de HTML5.
Recuerda que es importante entender la diferiencias entre ellas.

Etiqueta ul+li

Si listamos personas nominadas a los Oscars, dado que el orden no altera el significado.
debemos usar ul.

  • David Verdaguez
  • Jesús Castro
  • Israel Elejalde
  • Dani Rovira

Etiqueta ol+il

En el caso de que estemos ñistando elementos donde el orden es importante, como por
ejemplola clasificación de un mundial de futbol, debemos usar ol.

  1. España
  2. Paises Bajos
  3. Alemania
  4. Uruguay
